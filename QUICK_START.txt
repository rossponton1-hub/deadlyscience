🎯 KEY FEATURES
─────────────────────────────────────────────────────────────────────
1. Mode-Aware Links
   • In-person missing data → "Open Session" link
   • Virtual missing data → Survey action buttons only

2. Smart Scroll
   • ?edit=counts → Demographics section
   • ?edit=facilitator → Presenter section
   • ?edit=all → First editable section
   • Invalid param → Gracefully ignored

3. Conditional Navigation
   • From Data Completeness → Back link shows
   • Direct access → Back link hidden
   • Referrer-based detection

4. Session Finder
   • Browse 6 demo scenario cards
   • Navigate to any Deliver page
   • Clear prototype labeling

💡 PRO TIPS
─────────────────────────────────────────────────────────────────────
• Test in incognito/private mode (clean referrer state)
• Mobile test: Chrome DevTools → Toggle device toolbar → 360px
• Invalid params are OK: ?edit=xyz won't break anything
• Back link detection: Check document.referrer in console
• Multiple edit params: First one wins (?edit=counts&edit=all)
